Features

This plugin was built to make creating a slideshow on your website extremely simple and painless. By including the few files provided in this package, you'll be up and running within minutes.

  • Auto-sliding
  • Image titles & captions
  • Flexible configuration

Documentation

Firstly, you'll need to reference the SnapMotion JavaScript and CSS files within your <head> tag.

						<script src="js/jquery.snapmotion.js"></script>
						<link rel="stylesheet" href="css/snapmotion.css" type="text/css" />

Create your slides using a <ul> with a <li> for each slide. Each <li> should include the slide <img /> and a <span> tag containing any text you'd like to display on the slide.

For example:

						<ul id="slideshow">
							<li><img src="img/test_image_1.jpg" alt="Test Image 1" width="960" height="400" /></li>
							<li><img src="img/test_image_2.jpg" alt="Test Image 2" width="960" height="400" /></li>
							<li><img src="img/test_image_3.jpg" alt="Test Image 3" width="960" height="400" /></li>
							<li><img src="img/test_image_4.jpg" alt="Test Image 4" width="960" height="400" /></li>
							<li><img src="img/test_image_5.jpg" alt="Test Image 5" width="960" height="400" /></li>
						</ul>

Finally, call the SnapMotion() method on your slideshow. For example:

						$("ul#slideshow").SnapMotion();

Optional parameters include:

  • slideWidth (the width of a slide)
  • cIndent (the indentation of the 'controls' for the slideshow, in pixels, from the left of each slide)
  • fadeLength (the length, in milliseconds, of each slide transition)
  • ssDelay (the length, in milliseconds, between each slide transition)
						<script type="text/javascript">
							$(document).ready(function(){
								$("ul#slideshow").SnapMotion({
									'slideWidth'	: 960,
									'cIndent'		: 640,
									'fadeLength'	: 375,
									'ssDelay'		: 7500
								});
							});
						</script>
